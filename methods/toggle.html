w2layout.toggle

toggle(type, [immediate])

パネルの表示/非表示を切り替えます。

type : string
パネル名。
immediate : boolean
アニメーションを無効にするかどうか。オプションです。

説明

第1引数のtypeは、パネルの名前を指定します。 名前は以下を指定することができます。

  • top
  • left
  • main
  • right
  • preview
  • bottom

パネルが隠れている時に.toggle()を呼び出すと.show()メソッドを呼び出し、 表示されている時に.toggle()を呼び出すと、.hide()メソッドを呼び出します。

以下に例を示します。

$('#layout').w2layout({
    name    : 'layout',
    panels  : [
        { type: 'top', size: 40 },
        { type: 'main', content: 'This is main panel' },
        { type: 'preview', size: 200 }
    ]
});
 
// アニメーションあり。
w2ui['layout'].toggle('preview');
 
// アニメーション無し。
w2ui['layout'].toggle('preview', true);

デモ

レイアウトをクリックするたびに、previewパネルのを表示/非表示します。

JavaScript

$('#layout').w2layout({
    name    : 'mylayout',
    panels  : [
        { type: 'top', content: 'top', size: 40 },
        { type: 'main', content: 'This is main panel' },
        { type: 'preview', content: 'preview', size: 200 }
    ]
});
$('#layout').on('click', function() {
    w2ui['mylayout'].toggle('preview');
});
