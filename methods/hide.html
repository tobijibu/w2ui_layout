w2layout.hide

hide(panel, [immediate])

パネルを隠します。

panel : string
パネル名。
immediate : boolean
アニメーションを無効にするかどうか。オプションです。

説明

第1引数のpanelは、パネルの名前を指定します。 名前は以下を指定することができます。

  • top
  • left
  • main
  • right
  • preview
  • bottom

パネルは約0.2秒*1のアニメーションで隠れます。 アニメーションを無効化したい場合は、 第2引数のimmediateをtrueにしてください。

*1 公式では約2秒となっておりますが、ソースでは0.2秒になっています。


レイアウトを以下のように指定した場合、

$('#layout').w2layout({
    name    : 'layout',
    panels  : [
        { type: 'top', size: 40 },
        { type: 'main', content: 'This is main panel' },
        { type: 'preview', size: 200 }
    ]
});

アニメーション有りでパネルを隠します。

w2ui['layout'].hide('preview');

または、アニメーション無しでパネルを隠します。

w2ui['layout'].hide('preview', true);

デモ

レイアウトをクリックすると、previewパネルが隠れます。

JavaScript

$('#layout').w2layout({
    name    : 'mylayout',
    panels  : [
        { type: 'top', content: 'top', size: 40 },
        { type: 'main', content: 'This is main panel' },
        { type: 'preview', content: 'preview', size: 200, hidden: true }
    ]
});
$('#layout').on('click', function() {
    w2ui['mylayout'].hide('preview');
});
